ScanPasteFill

Edge Sidebar Autofill for Structured Scan & Paste Data

ScanPasteFill is a Microsoft Edge sidebar extension designed to simplify data entry for internal pharmacy and shipping workflows. It listens for structured scan or paste input, parses it into tokens, and automatically fills recognized web form fields.

How It Works

  1. Open the sidebar and scan or paste structured text (for example: \John|Doe\).
  2. ScanPasteFill parses the tokens and fills the matching form fields in the current tab.
  3. Profiles can be imported or edited directly within the sidebar.

Download & Install

This extension is distributed as an Unlisted Edge Add-on for internal use. If you have the direct link from your administrator, you can install it from the Microsoft Edge Add-ons Store.
